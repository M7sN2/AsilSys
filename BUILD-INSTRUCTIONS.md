# تعليمات بناء ملف التثبيت - نظام أسيل

## الطريقة السريعة (موصى بها)

### على Windows:
1. انقر نقراً مزدوجاً على ملف `build-installer.bat`
2. انتظر حتى يكتمل البناء
3. ملف التثبيت سيكون في مجلد `dist\` باسم `أسيل-Setup-1.0.0.exe`

---

## الطريقة اليدوية

### المتطلبات:
- Node.js مثبت على الجهاز
- npm مثبت

### الخطوات:

1. **فتح Terminal/PowerShell في مجلد المشروع**

2. **تثبيت التبعيات (إذا لم تكن مثبتة):**
   ```bash
   npm install
   ```

3. **بناء ملف التثبيت:**
   ```bash
   npm run build:installer
   ```

4. **الانتظار حتى يكتمل البناء** (قد يستغرق بضع دقائق)

5. **العثور على ملف التثبيت:**
   - المجلد: `dist\`
   - الاسم: `أسيل-Setup-1.0.0.exe`

---

## ملاحظات مهمة:

1. **حجم الملف:** سيكون حوالي 150-200 ميجابايت (يحتوي على Electron كامل)

2. **الوقت المستغرق:** قد يستغرق 5-10 دقائق حسب سرعة الجهاز

3. **الميزات المضمنة:**
   - ✅ جميع الملفات والأصول (assets)
   - ✅ قاعدة بيانات SQLite محلية
   - ✅ يعمل بدون إنترنت تماماً
   - ✅ يوميات التحصيل
   - ✅ جميع الشاشات والوظائف

4. **الأخطاء المحتملة:**
   - إذا ظهرت رسالة خطأ بخصوص `better-sqlite3`، قم بتشغيل:
     ```bash
     npm run rebuild
     ```
   - ثم حاول مرة أخرى:
     ```bash
     npm run build:installer
     ```

5. **التحقق من البناء:**
   - تأكد من وجود ملف `أسيل-Setup-1.0.0.exe` في مجلد `dist\`
   - حجم الملف يجب أن يكون أكبر من 100 ميجابايت
   - تأكد من أن الملف قابل للتشغيل

---

## توزيع ملف التثبيت:

1. انسخ ملف `أسيل-Setup-1.0.0.exe` من مجلد `dist\`
2. يمكنك نسخه على USB أو رفعه على Google Drive
3. على جهاز العميل، انقر نقراً مزدوجاً على الملف للتثبيت
4. اتبع خطوات التثبيت (اختر مجلد التثبيت، إلخ)

---

## معلومات إضافية:

- **اسم التطبيق:** أسيل
- **الإصدار:** 1.0.0
- **نوع الملف:** Windows Installer (NSIS)
- **البنية:** x64 (64-bit)
- **نظام التشغيل المطلوب:** Windows 10 أو أحدث

---

## استكشاف الأخطاء:

### المشكلة: "electron-builder not found"
**الحل:**
```bash
npm install --save-dev electron-builder
```

### المشكلة: "better-sqlite3 build failed"
**الحل:**
```bash
npm run rebuild
npm run build:installer
```

### المشكلة: "Icon file not found"
**الحل:** تأكد من وجود ملف `build/icon.ico`
